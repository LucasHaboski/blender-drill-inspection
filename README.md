# Blender Drill Inspection

## Objetivo
Automatizar a geração de imagens de referência para o controle de qualidade de brocas odontológicas. O projeto utiliza Python integrado à API do Blender para simular um ambiente de estúdio fotográfico real.

## Funcionalidades
- **Rotação Automática:** Script Python que rotaciona o modelo 3D da broca a cada 6 graus.
- **Renderização Realista:** Configuração de materiais e iluminação no Blender para mimetizar a "garra" e a câmera do maquinário real.
- **Dataset de Validação:** Geração de 60 imagens por ciclo para estabelecer o "Modelo Perfeito" de aprovação.

## Tecnologias
- Python 3.13.7
- Blender API (bpy)
- Modelagem 3D (Brocas Odontológicas)
